# Coin Chain

Coin Chain is a small desktop strategy/puzzle game built with Electron, Vite, and TypeScript.

The game is centered around placing and upgrading tiles on a grid, growing your economy, and pushing through day-based progression with increasing difficulty.

## Stack

- Electron desktop wrapper
- Vite renderer
- TypeScript codebase

## Run

```powershell
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
npm.cmd run dist:win
```

## Notes

- Main game source lives in `src/`
- Electron desktop entry code lives in `electron/`
- Packaged Windows builds can be produced as a portable `.exe`
